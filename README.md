# To-Do List App

A simple To-Do List built using HTML, CSS, and JavaScript.

## Features include:
- Add tasks
- Delete tasks
- Mark tasks as completed
- Drag & drop reordering
- Saves to localStorage

---


---
##  How It Works

- Tasks are created dynamically in the DOM  
- `localStorage` is used to persist tasks  
- Drag-and-drop uses native browser events  
- All events are handled using JavaScript

---


## How to Run

You don't need to install anything.

Just:

1. Download or clone the repo  
2. Open `index.html` in your browser  
3. That's it!
